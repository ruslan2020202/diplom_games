Задачи:
1. Подключить базу данных (и под нее сервер)
2. Сделать рейтинг лучший игроков

Сделано:
1. Сделан единый сервер (теперь не надо подключать сервер фронтенда и бэкенда отдельно, все
лежит на одном сервере)
2. Подключена база данных, в которой хранятся игроки и их результаты
3. Сделан рейтинг лучших игроков (сделан алгоритм для каждой игры (где присутствуют очки),
который собирает максимальные результаты игроков и делает из них отсортированный рейтинг)
4. Сделана форма логина, для нее и формы регистрации сделаны валидации(проверка почты
или почты и пароля соответственно)

Требования:
1. Python 3.10 и выше

Инструкция:
1. Установить все зависимости: pip install -r requirements.txt
2. Запустить сервер: python3 (если windows то python) app.py
3. Перейти на http://127.0.0.1:5000
ВСЕ
